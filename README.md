# Cybersecurity Bootcamp Portfolio - Week 1

### Author: MIJA TANJUNG

## Project: Python Advanced Port Scanner

### Deskripsi
Project ini adalah bagian dari Bootcamp Cybersecurity Week 1 ("The Toolmaker"). 
Script ini dibuat menggunakan Python Socket Programming untuk mensimulasikan cara kerja Network Scanner.

### Fitur Utama
- **Multi-Port Scanning:** Memindai daftar port krusial (21, 22, 80, 443, dll).
- **Banner Grabbing:** Mengambil informasi versi layanan yang berjalan di target.
- **HTTP Trigger:** Mampu memancing HTTP Server untuk memberikan respon banner.
- **Error Handling:** Menangani timeout dan koneksi yang ditolak dengan rapi.
